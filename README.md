# <h1 align="center"> 📊 Project - Job Analysis in Excel </h1>

<h2 align="center">  🔎 Introduction </h2>
The following projects were created to demonstrate Excel proficiency after completing Luke Barousse’s Excel for Data Analytics course. They focus on practical data analysis tasks commonly used in data-related roles.

### 1. Excel Salary Dashboard
![Excel Dashboard](/Gifts/Project_1/Excel%20Dashboard.png)

An interactive dashboard built in **Excel** to analyze salary trends across different data roles. It allows users to explore compensation patterns by **Job Title**, **Country**, and **Job Type (Full-time, Part-time, etc.)**. The dashboard also highlights the platform with the highest number of job postings and displays the total job count based on the selected criteria. 

[- Check out the project analysis here -](/Project_1/)


### 2. Job Analysis using Power Query and Power Pivot
![Excel Dashboard](/Gifts/Project_2/Excel%20Power%20Query%20Pivot.png)

A data analysis project that uses **Power Query** for data transformation and **Power Pivot** for data modeling. The objective is to analyze job market data for data-related roles, enabling deeper insights through structured data preparation and relational modeling. The analysis allows filtering by **Country** and **Job Title** to explore patterns and trends in the job market.
